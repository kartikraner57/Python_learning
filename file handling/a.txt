| Mode | **What it does** (Marathi + English)                                                                   |
| ---- | ------------------------------------------------------------------------------------------------------ |
| `r`  | **Read mode** – फाईल वाचण्यासाठी. File **आधीपासून असणं गरजेचं आहे**. नसेल तर **error येतो**.           |
| `w`  | **Write mode** – नवीन डेटा लिहायला. जुना डेटा **डिलिट होतो**. File नसल्यास, **create होते**.           |
| `a`  | **Append mode** – जुना डेटा **राखून ठेवतो**, नवीन डेटा शेवटी जोडतो. File नसल्यास, **create होते**.     |
| `r+` | **Read + Write mode** – वाचायला आणि लिहायला. जुना डेटा डिलिट होत नाही. फाईल आधीपासून असली पाहिजे.      |
| `w+` | **Write + Read mode** – लिहायला आणि वाचायला. जुना डेटा **डिलिट होतो**. File create होते जर नसेल.       |
| `a+` | **Append + Read mode** – जुना डेटा **राहतो**, नवीन शेवटी जोडतो. फाईल वाचण्यासाठी आणि लिहिण्यासाठी.     |
| `x`  | **Exclusive creation** – नवीन फाईल **create करतो**, जर फाईल आधीपासून असेल तर **FileExistsError** येतो. |

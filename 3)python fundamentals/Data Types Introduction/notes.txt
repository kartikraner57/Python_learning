Data types =>
              A data type in Python refers to the classification or categorization of data, which indicates what kind of data it is and what operations can be performed on it.

Examples of common data types in Python include:

int (Integer): Whole numbers (e.g., 1, -5)
float (Floating-point): Decimal numbers (e.g., 3.14, -0.01)
str (String): Sequence of characters (e.g., "hello")
bool (Boolean): Represents True or False values (e.g., True, False)
